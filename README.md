#Nest JS Build

### Project Start
```
1. git clone

2. cd guro-server-nest

3. npm install

4. root/src/config에 env 폴더 생성

5. 생성한 폴더 env 안에 .development.env 파일 추가 하기
-> root/src/config/env/.development.env

6. 파일 안에 정보 넣기

7. npm run start:dev

```
